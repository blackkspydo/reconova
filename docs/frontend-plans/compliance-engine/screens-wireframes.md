# Screens & Wireframes (Compliance Engine)

Scope: ASCII wireframes for all compliance screens, covering every state (default, loading, error, empty) with conditional rendering rules. Includes tenant compliance dashboard, assessment detail, report download, framework requests, and super admin framework/control/mapping management.

---

## Route Structure

| Route | Access | Description |
|-------|--------|-------------|
| `/compliance` | Tenant Owner (edit), Members (view) | Compliance dashboard |
| `/compliance/frameworks/{id}` | Tenant Owner / Members | Framework detail + assessment history |
| `/compliance/assessments/{id}` | Tenant Owner / Members | Assessment detail + control results |
| `/admin/compliance` | Super Admin | Framework management list |
| `/admin/compliance/frameworks/new` | Super Admin | Create framework form |
| `/admin/compliance/frameworks/{id}` | Super Admin | Framework detail (controls, mappings, tier access) |
| `/admin/compliance/requests` | Super Admin | Framework request review |
